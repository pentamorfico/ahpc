# Benchmarking Scripts

This folder contains scripts for performance measurement and comparison across implementations.

## Files

- `benchmark.py` - Basic benchmarking script
- `clean_benchmark.py` - Clean benchmarking with proper statistics
- `comprehensive_benchmark.py` - Complete performance analysis
- `final_benchmark.py` - Final comprehensive benchmarks
- `ultimate_benchmark.py` - Ultimate performance testing
- `typed_numba_benchmark.py` - Specific Numba benchmarking

## Usage

Run comprehensive benchmarks:
```bash
python comprehensive_benchmark.py
```

## Output

- Performance measurements across system sizes
- Statistical analysis with error bars
- Comparison tables and CSV exports
- Scaling behavior analysis
