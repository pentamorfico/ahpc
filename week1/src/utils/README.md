# Utility Scripts

This folder contains plotting and visualization utilities.

## Files

- `create_enhanced_plots.py` - Enhanced visualization creation
- `create_performance_plots.py` - Performance plotting utilities
- `generate_assignment_plots.py` - Assignment-specific plots
- `fix_assignment_plots.py` - Plot fixing and regeneration

## Usage

Generate all plots:
```bash
python create_enhanced_plots.py
```

Generate assignment plots:
```bash
python generate_assignment_plots.py
```

## Output

- PNG visualization files
- Performance comparison plots
- Technical analysis charts
- Assignment-specific figures
