# Analysis Scripts

This folder contains profiling and performance analysis tools.

## Files

- `bottleneck_analysis.py` - Bottleneck identification
- `comprehensive_profiling.py` - Complete profiling analysis
- `detailed_profiling.py` - Detailed function-level profiling
- `validation_analysis.py` - Results validation and verification

## Usage

Profile an implementation:
```bash
python comprehensive_profiling.py Water_sequential.py
```

## Output

- Function-level runtime distribution
- Memory usage analysis
- Cache performance metrics
- Bottleneck identification reports
